# ARF Brand Guidelines

Welcome to the official **Agentic Reliability Framework (ARF) Brand** repository.  
This folder contains all approved logos, typography, colors, and visual assets for ARF. Use these resources to maintain a consistent, professional, and enterprise-ready appearance across all platforms and communications.

---

## 1. Logos

### Primary Logo
- **File:** `Logo/ARF_Primary.svg`
- **Usage:** Main logo for all official documents, presentations, and the website.
- **Notes:** Maintain clear space equal to the height of the "A" in ARF on all sides. Do not stretch or distort.

### Symbol-Only Logo
- **File:** `Logo/ARF_Symbol.svg`
- **Usage:** Favicons, social media icons, or space-constrained placements.

### Favicon
- **File:** `Logo/ARF_Favicon.ico`
- **Usage:** Website tabs, browser bookmarks.

### PNG Variants
- **Files:** 512x512, 1024x1024, and transparent backgrounds included.
- **Usage:** Social media, slide decks, promotional materials.

---

## 2. Colors

| Role               | Hex       | RGB             | Usage                                      |
|-------------------|-----------|----------------|-------------------------------------------|
| Primary Blue       | #0055FF  | 0, 85, 255      | Main logo, buttons, highlights            |
| Secondary Dark     | #1A1A1A  | 26, 26, 26      | Text, backgrounds                          |
| Accent Gradient    | #00CFFF → #0055FF | 0, 207, 255 → 0, 85, 255 | Optional banners, highlights             |
| Neutral Gray       | #F5F5F5  | 245, 245, 245   | Backgrounds, overlays                      |

**Best Practices:**  
- Always maintain sufficient contrast for accessibility.  
- Use primary blue for brand emphasis; avoid substituting with other colors.

---

## 3. Typography

| Type           | Font Family       | Weight | Usage                          |
|----------------|-----------------|--------|--------------------------------|
| Headings       | Inter, sans-serif| Bold   | Titles, slide headers          |
| Body Text      | Inter, sans-serif| Regular| Paragraphs, labels             |
| Code / Terminal| Fira Code        | Regular| CLI, examples, screenshots     |

**Spacing & Scaling:**  
- Headings: 1.2x line-height  
- Body text: 1.5x line-height  
- Maintain consistent horizontal and vertical padding when combining logos and text.

---

## 4. Logo Usage Guidelines

### Do:
- Use official logo files only.
- Maintain minimum clear space around logos.
- Respect original colors and proportions.

### Don’t:
- Stretch, distort, or rotate logos.
- Place logos on clashing backgrounds without sufficient contrast.
- Alter colors or typography of the logo.

## 6. Example Layouts

- Slide decks: Use primary logo top-left or top-right with clear space.  
- Website: Use full-width banner or symbol-only favicon in headers.  
- Social posts: Prefer centered logo and correct banner aspect ratio.
